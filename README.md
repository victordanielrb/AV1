# AeroCode (AV1)

Projeto em TypeScript para simular a gestão de produção de aeronaves, com cadastro e controle de aeronaves, funcionários, peças, etapas e testes.

## Pré-requisitos

- Node.js: 20.x (LTS recomendado)
- npm: 10+

## Como rodar

1. Instale as dependências:

```bash
npm install
```

2. Execute em modo desenvolvimento:

```bash
npm run dev
```

3. Ou gere o build e rode a versão compilada:

```bash
npm run build
npm start
```
